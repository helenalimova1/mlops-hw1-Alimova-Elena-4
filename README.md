Цель проекта

Цель данного проекта — разработка и обучение модели машинного обучения для классификации данных. Используется пайплайн DVC для управления данными и моделями, а также MLflow для логирования экспериментов.



Как запустить

Для запуска проекта выполните следующие команды:



git clone <repo>  # Замените <repo> на URL вашего репозитория

cd <repo>

pip install -r requirements.txt

dvc pull  # Загрузка данных и артефактов

dvc repro  # Воспроизведение пайплайна



Краткое описание пайплайна

Пайплайн состоит из двух основных стадий:



1\. \*\*prepare\*\*: Подготовка данных, включая разделение на обучающую и тестовую выборки, базовую очистку и сохранение обработанных данных.

2\. \*\*train\*\*: Обучение модели с использованием выбранного алгоритма (например, логистической регрессии или случайного леса).



Каждая стадия определена в файле dvc.yaml, что позволяет легко управлять процессом обработки данных и обучения модели.



Где смотреть UI MLflow

Для просмотра интерфейса MLflow поднимите локальный сервер с помощью следующей команды:



bash

mlflow ui --backend-store-uri sqlite:///mlflow.db

После этого откройте веб-браузер и перейдите по адресу http://localhost:5000, чтобы увидеть логирование параметров, метрик и артефактов ваших экспериментов.



Замените <repo> на фактический URL вашего репозитория перед использованием.
